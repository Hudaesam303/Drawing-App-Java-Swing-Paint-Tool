🎨 Drawing App (Java Swing)

A simple desktop painting application built using Java Swing. It allows users to draw shapes, use different tools, and customize their drawings with colors and styles.

✨ Features
🟦 Draw shapes (Rectangle, Oval, Line)
✏️ Free-hand drawing tool
🧽 Eraser tool
🎨 Color selection (preset colors + custom color picker)
📏 Adjustable stroke size
🔲 Filled or outline shapes
➰ Dotted or solid lines
↩️ Undo last action
🗑 Clear canvas
💾 Save drawing as PNG image
📂 Open images and edit them
🛠 Technologies Used
Java
Swing (GUI)
AWT
ImageIO
🎯 Project Purpose

This project was built to practice:

Object-Oriented Programming (OOP)
Event handling in Java
GUI development using Swing
Basic 2D graphics rendering
